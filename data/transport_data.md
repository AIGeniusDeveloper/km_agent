# Transport - Base de Connaissances

## Entretien Véhicules

### Maintenance Préventive
- **Vidange Huile Moteur** : Tous les 5000-10000 km. Utiliser huile recommandée par constructeur
- **Filtre à Air** : Nettoyer tous les 5000 km, remplacer tous les 20000 km
- **Pneus** : Vérifier pression hebdomadaire. Rotation tous les 10000 km
- **Freins** : Vérifier plaquettes tous les 20000 km. Remplacer si <3mm

### Contrôles Quotidiens (Véhicules Commerciaux)
- **Niveaux** : Huile moteur, liquide de refroidissement, lave-glace
- **Pneumatiques** : Pression, usure, objets coincés
- **Éclairage** : Phares, clignotants, feux stop
- **Freins** : Test de freinage avant départ

### Pannes Courantes
- **Batterie Déchargée** : Câbles de démarrage, vérifier alternateur
- **Surchauffe** : Vérifier niveau liquide refroidissement, radiateur, durites
- **Crevaison** : Roue de secours, cric, clé en croix

## Conduite Économique

### Techniques d'Éco-Conduite
- **Anticipation** : Regarder loin, éviter freinages brusques
- **Régime Moteur** : Passer les vitesses tôt (2000-2500 tr/min diesel, 2500-3000 essence)
- **Vitesse Constante** : Utiliser régulateur si disponible
- **Climatisation** : Limiter l'usage (augmente consommation de 10-20%)

### Économies de Carburant
- **Pression Pneus** : Pneus sous-gonflés = +5% consommation
- **Charge** : Enlever poids inutiles du véhicule
- **Aérodynamisme** : Retirer galerie de toit si non utilisée
- **Entretien** : Moteur bien entretenu = -10% consommation

## Sécurité Routière

### Équipements Obligatoires
- **Triangle de Signalisation** : Placer à 30m en cas de panne
- **Gilet Réfléchissant** : Porter dès la sortie du véhicule
- **Extincteur** : Vérifier pression annuellement
- **Trousse de Secours** : Pansements, désinfectant, compresses

### Prévention Accidents
- **Distance de Sécurité** : 2 secondes minimum (4 secondes si pluie)
- **Angles Morts** : Vérifier avant changement de voie
- **Fatigue** : Pause 15 min toutes les 2h
- **Alcool** : 0 tolérance. Taux légal = 0.5g/L mais risque dès 0.2g/L

### Conduite Défensive
- **Prévoir** : Anticiper les erreurs des autres usagers
- **Visibilité** : S'assurer d'être vu (feux, klaxon si nécessaire)
- **Échappatoires** : Toujours avoir une issue de secours mentale

## Transport Public

### Gestion de Flotte
- **Planning Maintenance** : Calendrier préventif pour chaque véhicule
- **Carnet de Bord** : Consommation, pannes, réparations
- **Rotation** : Alterner véhicules pour usure uniforme

### Optimisation des Trajets
- **Itinéraires** : Minimiser distances à vide
- **Horaires** : Adapter aux heures de pointe
- **Remplissage** : Maximiser le taux d'occupation

## Motos et Deux-Roues

### Sécurité Spécifique
- **Casque** : Obligatoire, homologué, bien ajusté
- **Visibilité** : Vêtements clairs/réfléchissants
- **Position** : Visible dans rétroviseurs des voitures

### Entretien Moto
- **Chaîne** : Graisser tous les 500 km, tension correcte
- **Freins** : Vérifier liquide et plaquettes mensuellement
- **Suspension** : Vérifier amortisseurs, fourche

## Vélos et Transport Non Motorisé

### Entretien Vélo
- **Chaîne** : Nettoyer et graisser mensuellement
- **Freins** : Vérifier patins, câbles
- **Pneus** : Pression optimale (indiquée sur flanc)
- **Éclairage** : Obligatoire la nuit (avant blanc, arrière rouge)

### Sécurité Cycliste
- **Casque** : Fortement recommandé
- **Signalisation** : Bras tendus pour indiquer direction
- **Prudence** : Rouler à 1m du bord, éviter portières

## Logistique

### Chargement
- **Répartition** : Poids équilibré, centre de gravité bas
- **Arrimage** : Sangles, filets. Vérifier tous les 100 km
- **Limites** : Respecter PTAC (Poids Total Autorisé en Charge)

### Documentation
- **Permis de Conduire** : Catégorie adaptée au véhicule
- **Carte Grise** : À jour, dans le véhicule
- **Assurance** : Minimum responsabilité civile
- **Contrôle Technique** : Selon réglementation locale

## Routes Difficiles

### Conduite en Saison des Pluies
- **Vitesse Réduite** : Diviser par 2 sur route mouillée
- **Aquaplaning** : Si perte d'adhérence, lever le pied, ne pas freiner
- **Visibilité** : Allumer feux même de jour

### Pistes Non Goudronnées
- **Vitesse Adaptée** : Ralentir dans ornières, nids-de-poule
- **4x4** : Engager avant zones difficiles
- **Enlisement** : Dégonfler légèrement pneus, placer branches sous roues

## Coûts d'Exploitation

### Calcul Coût au Kilomètre
- **Fixes** : Assurance, taxes, amortissement
- **Variables** : Carburant, entretien, pneus
- **Formule** : (Coûts annuels) / (Km parcourus)

### Optimisation
- **Groupage** : Combiner plusieurs livraisons
- **Retour Chargé** : Éviter trajets à vide
- **Négociation** : Carburant, pièces en gros
