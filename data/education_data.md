# Éducation & Formation - Base de Connaissances

## Pédagogie Active

### Méthodes d'Enseignement
- **Apprentissage par Projet** : Les élèves travaillent sur des projets concrets. Favorise l'autonomie et la créativité.
- **Classe Inversée** : Leçons à la maison (vidéos), exercices en classe. Maximise le temps d'interaction.
- **Apprentissage Collaboratif** : Travail en groupe, peer teaching. Développe les compétences sociales.

### Gestion de Classe
- **Règles Claires** : Établir 3-5 règles simples dès le début. Les afficher visiblement.
- **Routine** : Commencer chaque cours de la même manière (rituel). Réduit l'anxiété.
- **Renforcement Positif** : Féliciter les bons comportements plutôt que punir les mauvais.

## Évaluation

### Types d'Évaluation
- **Formative** : Pendant l'apprentissage (quiz, questions orales). Permet d'ajuster l'enseignement.
- **Sommative** : En fin de séquence (examen). Mesure les acquis.
- **Diagnostique** : En début d'année. Identifie le niveau initial.

### Critères d'Évaluation
- **SMART** : Spécifique, Mesurable, Atteignable, Réaliste, Temporel
- **Grille de Critères** : Définir clairement ce qui est attendu (ex: présentation orale = contenu 40%, expression 30%, support visuel 30%)

## Différenciation Pédagogique

### Adaptation aux Niveaux
- **Groupes de Besoin** : Regrouper les élèves par niveau pour certaines activités
- **Tutorat par les Pairs** : Les élèves avancés aident les autres
- **Supports Variés** : Textes, vidéos, manipulations pour toucher tous les profils

### Inclusion
- **Élèves à Besoins Spéciaux** : Adapter les supports (gros caractères, audio)
- **Multilinguisme** : Utiliser la langue maternelle comme pont vers la langue d'enseignement

## Ressources Pédagogiques

### Matériel de Base
- **Tableau Noir** : Utiliser des couleurs pour structurer (titres en rouge, exemples en vert)
- **Affiches Pédagogiques** : Alphabet, tables de multiplication, cartes
- **Matériel Manipulable** : Bâtonnets, cailloux pour les mathématiques

### Ressources Numériques (si disponibles)
- **Khan Academy** : Vidéos éducatives gratuites
- **Wikipedia** : Encyclopédie collaborative
- **Projets Open Source** : Manuels libres, exercices

## Formation des Enseignants

### Développement Professionnel
- **Observation Mutuelle** : Les enseignants s'observent entre eux
- **Communautés de Pratique** : Groupes d'échange entre enseignants
- **Formation Continue** : Ateliers sur les nouvelles méthodes

### Gestion du Stress
- **Préparation** : Planifier les cours à l'avance réduit le stress
- **Pause** : Prendre des pauses régulières pendant la journée
- **Réseau de Soutien** : Échanger avec des collègues

## Contexte Africain

### Classes Surchargées
- **Moniteurs** : Utiliser des élèves avancés comme assistants
- **Travail en Groupe** : Réduire le nombre d'interactions individuelles nécessaires
- **Affichages** : Maximiser l'utilisation des murs pour l'information permanente

### Ressources Limitées
- **Matériel Local** : Utiliser ce qui est disponible (sable pour écrire, bâtons pour compter)
- **Partage** : Un manuel pour 3-4 élèves
- **Créativité** : Fabriquer du matériel pédagogique (cartes flash, jeux)

### Multilinguisme
- **Transition Progressive** : Commencer en langue locale, introduire progressivement la langue officielle
- **Code-Switching** : Alterner entre langues pour clarifier les concepts difficiles
