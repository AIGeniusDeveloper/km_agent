# Environnement - Base de Connaissances

## Gestion des Déchets

### Tri des Déchets
- **Organiques** : Restes alimentaires, feuilles. → Compostage
- **Recyclables** : Plastique, verre, métal, papier. → Collecte séparée
- **Dangereux** : Piles, médicaments, produits chimiques. → Points de collecte spéciaux

### Compostage
- **Matériaux Verts** : Épluchures, herbe fraîche (azote)
- **Matériaux Bruns** : Feuilles sèches, carton (carbone)
- **Ratio** : 2/3 bruns, 1/3 verts
- **Durée** : 3-6 mois pour compost mature

### Réduction des Déchets
- **Refuser** : Dire non aux sacs plastiques, pailles
- **Réduire** : Acheter en vrac, éviter le suremballage
- **Réutiliser** : Bocaux, sacs en tissu
- **Recycler** : Dernier recours après les 3R précédents

## Reboisement

### Choix des Espèces
- **Espèces Locales** : Mieux adaptées au climat, résistantes
- **Espèces Utiles** : Fruits (manguier, avocatier), bois (acacia), médicinales
- **Diversité** : Planter plusieurs espèces pour résilience

### Plantation
- **Saison** : Début de saison des pluies
- **Espacement** : 3-5m entre arbres selon l'espèce
- **Trou** : 40x40x40cm, mélanger terre avec compost
- **Protection** : Clôture contre animaux, paillage

### Entretien
- **Arrosage** : Régulier la première année (10-20L/semaine)
- **Désherbage** : Rayon de 1m autour de l'arbre
- **Taille** : Enlever branches mortes, former la couronne

## Conservation de l'Eau

### Techniques Agricoles
- **Paillage** : Couvrir le sol (paille, feuilles). Réduit évaporation de 50%
- **Zaï** : Trous de plantation avec compost. Capture l'eau de pluie
- **Demi-Lunes** : Bassins en croissant. Concentrent l'eau

### Captage d'Eau de Pluie
- **Toits** : Gouttières → citernes. 1mm pluie = 1L/m² de toit
- **Bassins** : Creuser des bassins pour stocker l'eau de ruissellement
- **Barrages de Sable** : Dans lits de rivières saisonnières

## Lutte Contre l'Érosion

### Méthodes Végétales
- **Haies Vives** : Arbustes en courbes de niveau. Freinent le ruissellement
- **Bandes Enherbées** : Herbes le long des pentes
- **Agroforesterie** : Arbres + cultures. Racines stabilisent le sol

### Méthodes Mécaniques
- **Terrasses** : Aplanir les pentes en gradins
- **Cordons Pierreux** : Lignes de pierres en courbes de niveau
- **Fossés** : Canaux pour détourner l'eau

## Énergie Renouvelable

### Solaire
- **Cuiseurs Solaires** : Boîte isolée + vitre. Température 100-150°C
- **Séchoirs Solaires** : Pour fruits, légumes, poisson
- **Panneaux Photovoltaïques** : Électricité pour éclairage, pompage

### Biogaz
- **Digesteur** : Fumier animal + eau → méthane + engrais liquide
- **Dimensionnement** : 1 vache = 1m³ biogaz/jour = 2-3h cuisson
- **Avantages** : Énergie propre + fertilisant de qualité

### Biomasse
- **Foyers Améliorés** : Consomment 50% moins de bois que foyers traditionnels
- **Briquettes** : Déchets agricoles compressés. Alternative au charbon
- **Pyrolyse** : Transformation bois en charbon avec récupération de gaz

## Biodiversité

### Protection des Espèces
- **Corridors Écologiques** : Connecter les zones protégées
- **Zones Tampons** : Autour des aires protégées. Agriculture durable autorisée
- **Lutte Anti-Braconnage** : Patrouilles, sensibilisation communautaire

### Agriculture et Biodiversité
- **Polyculture** : Plusieurs espèces dans un même champ
- **Rotation** : Alterner les cultures chaque saison
- **Haies** : Refuge pour insectes utiles et oiseaux

## Changement Climatique

### Adaptation
- **Variétés Résistantes** : Sécheresse, chaleur, maladies
- **Diversification** : Ne pas dépendre d'une seule culture
- **Assurance Récolte** : Mécanismes de protection financière

### Atténuation
- **Séquestration Carbone** : Planter des arbres, améliorer les sols
- **Réduction Émissions** : Énergies renouvelables, transports propres
- **Gestion Durable** : Forêts, pêcheries, pâturages

## Sensibilisation

### Éducation Environnementale
- **Écoles** : Clubs environnement, jardins scolaires
- **Communautés** : Théâtre, radio, affiches
- **Démonstrations** : Parcelles modèles, visites d'échange

### Mobilisation
- **Journées d'Action** : Plantation d'arbres, nettoyage
- **Concours** : Villages/écoles les plus propres
- **Récompenses** : Reconnaître les champions de l'environnement
